<b>Notes on Basics of the Web and HTML</b>

World Wide Web

The web is made up of you, your computer and browser, the internet and servers. You use your browser to make requests via the internet to servers. The requests use a protocol called HTTP .

The internet is the world's largest computer network. Servers are computers which host files.

To learn more, you can watch THIS VIDEO which describes how the internet works in a little more detail.

HTML, or HyperText Markup Language, is made up of text content (what you see), markup (what it looks like), references to other documents (i.e. images and videos), and links to other pages.

Why Computers Are "Stupid"

Computers do exactly what they are told. As a result, a small mistake in code can cause large problems.

HTML Markup

Tags

Some important tags to learn are

  • bold: <b> </b>
  • italics: <em> </em>

Tags may have attributes. For example, the anchor tag which is used for making links has an attribute.

Note: a void tag (such as <br>) does not have any content, therefore it does not need a closing tag

Inline vs Block

An inline element only takes up as much width as necessary, and does not force line breaks.

A block element is an element that takes up the full width available, and has a line break before and after it.
